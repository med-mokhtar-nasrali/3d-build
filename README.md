# 3D Build - Real Estate Website

Real estate website with AI chatbot assistant.

## Quick Start

```bash
# Install and start everything
start-all.bat
```

Or manually:

```bash
# Backend
cd server
npm install
npm start

# Frontend (new terminal)
npm run dev
```

Visit `http://localhost:5173`

## Features

- Real estate property showcase
- Interactive floor plans
- AI chatbot (Aria) powered by GPT-120B
- Admin dashboard
- Contact forms
